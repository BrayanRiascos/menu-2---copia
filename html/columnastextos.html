Columnas de texto

Usando HTML mas CSS

Propieades a usar

Column count nos permite decidir en cuantas columnas queremos dividir en listado de item

column-count:2

Column grap, permite definir que espacio deseamos entre columnas 4px, 2px

column-gap:10px

Column rule, nos permite manipular tres caracteristicas que en ordens serian, grosor de las linea 4px, tipo de linea y por ultimo el color

column-rule:4px dotted gray;

Pasos para iniciar a usar texto el columnas:

Dentro de un DIV, haciendo uso de un identificador o clase, escribimos el listado de lo que deseamos este en columnas, en este caso ciudades de colombia, y dentro del DIV usamos la etiqueta < li> para poder listar los item que deseamos queden en columnas. 

<div id="columnas">
                <li>Cali</li>
                <li>Popayan</li>
                <li>Pasto</li>
                <li>Armenia</li>
                <li>Cartagena</li>
                <li>Cucota</li>
                <li>Barranquilla</li>
                <li>Bogota DC</li>
                <li>Pereira</li>
                <li>Santa Marta</li>	
                <li>Villavicencio</li>
                <li>Manizales</li>				
                <li>Medellin</li>
                <li>Neiva</li>
                <li>Yopal</li>
                <li>Riohacha</li>
           </div>

Una vez se tenga el listado de items a colocar en columnas, se hace uso del identificador o clase usado, en este caso seria el #columnas, este identificado se usa en la hoja de estilo.css el cual debe quedar de la siguiente manera:


#columnas{
            column-count:4;
            column-gap:20px;
            column-rule:4px dotted gray;
             }

Y como resutado final tenemos las siguientes columnas, con la respetiva informacion:

  • Cali
  • Popayan
  • Pasto
  • Armenia
  • Cartagena
  • Cucuota
  • Barranquilla
  • Bogota DC
  • Pereira
  • Santa Marta
  • Villavicencio
  • Manizales
  • Medellin
  • Neiva
  • Yopal
  • Riohacha

    • Recordemos que en la hoja de estilos podemos utilizar diferentes propiedades de diseño...

    #columnas{
            column-count:4;
            column-gap:20px;
            column-rule:4px double black;
           list-style: none;  
         }

    Tipos de lineas

    En lo referente a los tipos de separadores entre columnas de texto existen las siguientes posibilidades:

    none:sin línea separadora.
    solid: línea sólida.
    dotted: línea punteada.
    dashed: línea discontinua (segmentos cortos de línea).
    double: doble línea paralela.
    groove:línea en relieve 3D hacia adentro.
    ridge: línea en relieve 3D hacia afuera.
    inset: linea con el color en bajorelieve.
    outset: línea que simula el relieve.
