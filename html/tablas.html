Tablas en HTML

Las tablas es el elemento del lenguaje HTML que utilizaremos para mostrar información de forma agrupada. Ya sea texto, imágenes, vídeos,…

Para crear una tabla vamos a necesitar conocer, al menos, tres elementos: table, tr y td.

Mi tabla
table apertura de la tabla
tr Cada fila de la tabla está marcada con la esta etiqueta
td Cada celda de la tabla está marcada con esta etiqueta
th Celdas que sirven de cabecera para la fila o columna
caption Titulo de la tabla
