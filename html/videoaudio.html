Audio y video en HTML

El elemento video se utiliza para incrustar vídeos en un documento HTML

Formatos compatibles

  • MP4
  • OGG
  • WEBM

Atributos del elemento video

  • width: Indica la anchura del vídeo
  • height: Indica la altura del vídeo
  • controls: Establece si tendrá la barra de navegación
  • autoplay: Indica si el vídeo se inicia detenido o ya aparece por defecto reproduciéndose
  • loop: Establece si al acabar el vídeo, éste debe volver a empezar o no
  • muted: Indica si el audio se reproduce o se reproduce sin audio
  • preload: Indica si el vídeo se empieza a cargar cuando el navegador carga la página html
  • poster: Da la posibilidad de establecer una imagen (jpg, gif o png) como fotograma inicial del video

Audio en HTML

El elemento audio se usa para insertar contenido de audio en un documento HTML

Formatos compatibles

  • MP3
  • OGG
  • WAV

Atributos del elemento AUDIO

  • controls: Establece si tendrá la barra de navegación
  • autoplay: Indica si el audio se inicia automáticamente al cargar la página html
  • loop: Establece si al acabar el audio, éste debe volver a empezar indefinidamente
  • preload: Indica si el audio se empieza a cargar cuando el navegador carga la página html
