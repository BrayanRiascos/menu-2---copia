Listas HTML

Permite crear conjuntos de elementos en forma de lista dentro de una página, generalmente indicados, por un guión o número.


    Tipos de listas en HTML:

  • Listas ordenadas
  • Listas no ordenadas

Listas ordenadas

Nos muestran los elementos de la lista en orden. Para representar el orden tendremos los elementos numerados. Es decir, cada uno de los elementos irá precedido de un número o letra que establezca su orden.


Las listas en HTML ordenadas se representan mediante el elemento OL.

Ejemplo:

Resultado:

  1. Lunes
  2. Martes
  3. Miercoles
  4. Jueves
  5. Viernes
  6. Sabado
  7. Domingo

Listas desordenadas

Nos sirven para mostrar los elementos sin ningún tipo de orden, simplemente precedidos por una viñeta que puede ser un punto, un cuadrado,…


Para definir una lista desordenada en HTML utilizamos el elemento ul.

Ejemplo:

Resultado:

  • Rojo
  • Amarillo
  • Blanco
  • Naranja
