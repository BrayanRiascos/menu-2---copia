Modo responsive

Para hacer que nuestros sitios web se adapten a cualquier dispositivo, debemos utilizar @mediaqueries, los cuales se configuran en la hoja de estilos, juegan un papel muy importante para que los sitios web sean responsive.

Como funciona:

Podemos ajustar nuestro sitio web teniendo en cuenta las siguientes resoluciones estandares ajustables por medio de los pixles:


Resoluciones iguales o menores de 480 píxeles: para móviles
Resoluciones de 481px a 768 píxeles: para tablets
Resoluciones mayores de 768 píxeles: para ordenadores

Estructura de un media queri

@media only screen and (max-width:320px){
                    <!--Este mediaquerie se ejecutará sólo cuando la resolución horizontal sea menor de 320 píxeles-->
                  }
                  
                  @media only screen and (min-width:980px){
                    <!--Este mediaquerie se ejecutará sólo cuando la resolución horizontal sea mayor de 980 píxeles-->
                  }

Para que sirve

Nos permite programar que clases, etiquetas, identificadores se modifiquen dependiendo la resolucion y el dispositivo que este leyendo nuestro sitio web, podemos decir que en una relosucion de XXXpx, la letra disminuya, se ajusten cajas, cambie de color los fondos, se oculten ciertas cosas, aplicar efectos... Todo esto va de la mano de flexbox, que permite adecuar nuestros sitios web responsives.
